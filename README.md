# Affinity Simulator

This code was created and inspired by the following video
https://www.youtube.com/watch?v=tyQ-Yh1yfRA&t=213s&pp=ygUOaHlwZXJnZW9tYW5jZXLSBwkJTwsBhyohjO8%3D

in order to simulate games with the MtG deck Grixis Affinity and optimize the manabase.
This is not the official repository of the video and is not, in any way, affiliated with that.

It should also be noted that all the code here has been generated with the assistance of Copilot and might contain mistakes.
There are 2 apis from which one can import functions to run the games simulations. 
The first one is api, the second one is api_hidden_info. When using the first api the model has perfect foresight of how the deck is stacked, in the second case this does not happen.

Open this folder as a PyCharm project and select Python 3.14 as interpreter. The code is compatible with Python >=3.10 and can be used with Python 3.14.
